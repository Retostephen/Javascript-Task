JavaScript Exercises

This page contains three beginner JavaScript exercises that run in the browser using prompts and alerts.

Exercise 1: Initials Generator

Enter your full name and the program will generate your initials. For example, Andrew Solex becomes A.S.

Exercise 2: Word Position Finder

Enter a sentence and a letter, and the program will tell you the position of that letter in the sentence.

Exercise 3: Username Generator

Enter your first and last name, and the program will create a username made up of your full name in lowercase followed by the total number of characters. For example, Andrew Solex becomes andrewsolex_11.
